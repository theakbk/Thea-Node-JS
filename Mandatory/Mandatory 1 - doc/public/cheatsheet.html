Cheatsheet for commandline kommandoer

Kommando Beskrivelse
NMP install Her installeres NodeJS packet manager
Nodemon app.js Kører en applikation, med Nodemon, ved navn app.js
pwd Viser mig stien for min nuværende lokation
cd CD betyder "Change directory" og lader mig skifte lokation/directory til der hvor jeg gerne vil hen
ls -l Denne kommando lader mig se indholdet af den mappe jeg er i lige nu
cat Lader mig se indholdet af en bestemt fil
node app.js Lader mig kører en applikation ved navn app.js
npm run [script-name] Denne kommando lader mig kører en applikation med envirement variables som betingelser.
